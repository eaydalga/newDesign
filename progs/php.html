PHP Hyperlink Preprocessor

11 Ocak 2008

Php Programlama

"PHP Hypertext Preprocessor" cümlesinin ilk harflarinden oluşur. Kendi kendini tanımlayan açık kaynak yazılım kodlamasıdır (GNU gibi bir açılımı olan açık kaynak tanımlamasıdır). Genel amaçlı kullanımı, özellikle Web tasarımını değiştirmeden HTML içine gömülerek kullanılabilir. Örnek yazılım aşağıda anlatılmıştır: 
<html>
    <head>
        <title>Ornek</title>
    </head>
    <body>

        <?php 
        echo "Merhaba, Dünya!"; 
        ?>

    </body>
</html>
Burada değişik olan, diğer programlama dillerindekinden farklı olan, HTML kodlarını yazmak için uzun uzadıya kodlama yapılmamasıdır. HTML olduğu gibi kullanılır. PHP kodlaması özel başlama ve bitiş biçimleriyle tanımlanır. HTMl dosyası bu biçimlerin içinde PHP konumuna geçer.

PHP kodu, sunucu tarafında çalıştırılır. Eğer sunucunuzda yukarıdaki örnek gibi bir kodlama varsa tarayıcı yalnız HTML kodlarını görür. PHP kodları tarayıcılarda görünmediğinden güvenlik sorunu tarayıcı üzerinde çalışan kodlamalara pek benzemez.

Bu belgeler PHP öğrenilirken resmi PHP web sitesinde bulunan belgelerden kendi yazım biçimi ve sözlerimizle hazırlanmıştır. Daha çok bilgi edinmek için PHP Manual sayfalarının incelenmesini öneririz.

Burada kullanılan konu başlıkları resmi sitedeki konu başlıklarıyla aynıdır. Hatta örnekler bile aynıdır. Amaç resmi PHP el kitabıyla karşılaştırma yaparken uyum sorunu olmamasını sağlamaktır.
