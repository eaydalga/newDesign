com_create_guid

(PHP 5 de kullanılrı)

com_create_guid — Evrensel Tekil Kimlik (GUID) yaratır.

Açıklama

string com_create_guid ( void )

Generates a Globally Unique Identifier (GUID) and returns it as a string. A GUID is generated in the same way as DCE UUID's, except that the Microsoft convention is to enclose a GUID in curly braces. Evrensel Tekil Kimlik (GUID) yaratır ve onun yazı dizisini döndürür. GUID, DCE UUID gibi yaratılır.

uuid_create() işlevine de bakın. uuid PECL uzantısıdır.
