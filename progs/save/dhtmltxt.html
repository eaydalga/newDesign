WEB tasarımı yapmaya kalkınca DHTML ve DOM sözcüklerini duymamak elde değildir. Bunların neler olduğunu merak ettiyseniz kısaca söz edelim: DTML dinamik HTML sayfaları demektir. Sayfa tarayıcı tarafından yüklendikten sonra içeriğinin görüntüleniş biçimini değiştirmeye yarar.

WEB'de gördüğünüz pek çok sayfada DHTML etkilerinin neler olduğunu bilirsiniz. Bu sayfaların kodlarına baktığınızda DHTML kodunu anlamaktan vazgeçmiş olabilirsiniz. Hemen pes etmeyin. DHTML ne kolay hem de eğlencelidir. Aslında Javascript dili karışımıyla CSS ve HTML karışımıdır.

Buradaki eğitim sayfaları DHTML'in temel kavramlarını ve değişik özelliklerini içerir ve bunların nasıl kullanılacağını anlatmaya çalışır. Daha sonra W3C (World Wide WEB Consortium) DOM (Document Object Model) Özelliği bu sayfalarda anlatılır. W3C WEB sayfalarında kullanılan biçimler/komutlar hakkında tanımların yapıldığı birliktir. Döküman nesne modellemesi bir sayfadaki herşeye erişmenizi, ve değiştirmenizi, sağlar.

urada anlatılanlar ilk başlayanlara göre değildir. CSS, Javascript ve HTML konusunda bilginiz olduğu varsayılmıştır.
