Kodlama (script)

HTML sayfalarını daha dinamik ve etkileşimli yapmak için kodlama eklemek yararlıdır.

HTML sayfalrını daha canlı yapmak için sıklıkla kullanılan iki değişik kodlama (script) vardır. HTML içinde "javascript" ve "vbscript" zamanınız varsa öğrenilmesi gereken kodlama dilleridir.

HTML Javascript Kodu

Eğer HTML içinde "javascript" kullanmak isterseniz <script> biçimi kullanılır. "Javascript" kodlamasını daha ayrıntılı öğrenmek isterseniz bu sitede konuya ilişkin javascript açıklamalarını okumanızı sağlık veririz. WEB sayfalarınıza doğru biçimde "Javascript" kodu eklemek için aşağıdaki örnek tanımları inceleyin.

<script type="text/javascript"> <!--script ***Bazı javascript kodları buraya yazılır*** --> </script>
HTML içindeki kodlama <script> biçiyle tanımlanır. Eğer "type" niteliğini belirtmek gerekirse, kullanılan kodlama dilini de tanımlamalısınız. 
<html>
<head>
</head>
<body>

<script type="text/javascript">
document.write("Merhaba Dünya!")
</script>

</body>
</html>

HTML'de Vbscript

Web sayfalarına "vbscript" eklemek için yine "script" biçimi kullanılır. Aşağıda "vbscript" kodunu doğru eklemenin yöntemi gösterilir: 

<script type="text/vbscript">
<!--script
***vbscript kodu buraya yazılır***
-->
</script>

Vbscript kullanmak için "type" niteliğine "text/vbscript" yazmak gerekir. Aynı CSS tanımlamak gibidir. Vbscript kodları çevresinde mutlaka açıklama kullanılır. Burada vbscript desteklemeyen tarayıcılarda sorun çıkmaması ve tarayıcıdan vbscript kodunu ayırmak amacı vardır.

HTML "script" kodlama hakkında daha başka bilgiler için Script'ler sayfasını tıklayın.
