Resim Galerisi

CSS resim galerisi kurmak için de kullanılabilir.

Örnek galeri

Aşağıdaki kaynak kod CSS ile kurulan resim galerisi için bir örnektir.

Kaynak kod aşağıdakine benzer:
 
<pre>
<html>
<head>
<style type="text/css">
div.img
  {
  margin: 2px;
  border: 1px solid #0000ff;
  height: auto;
  width: auto;
  float: left;
  text-align: center;
  }
div.img img
  {
  display: inline;
  margin: 3px;
  border: 1px solid #ffffff;
  }
div.img a:hover img
  {
  border: 1px solid #0000ff;
  }
div.desc
  {
  text-align: center;
  font-weight: normal;
  width: 120px;
  margin: 2px;
  }
</style>
</head>
