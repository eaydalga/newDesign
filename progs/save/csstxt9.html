Liste (list)

Liste özelliği listenin madde işaretini değiştirmeye, ya da bir resmi bu işaret yerine kullanmaya yarar.

Genel

Liste özelliği listenin madde işaretini değiştirmeye, ya da bir resmi bu işaret yerine kullanmaya yarar.
Destekleyen Tarayıcılar:IE: Internet Explorer, F: Firefox, N: Netscape.
W3C: Buradaki sayısal değer CSS türünü (CSS1 ya da CSS2) gösterir.

Liste Stil (list-style) Özelliği

Bir tanımda tüm özellikleri belirtmek için kısaltılmış tanımdır. 
ul
{
list-style: disc outside
}

ol
{
list-style: decimal inside
}
Olası değerler:
liste-stil-tipi liste-stil-yeri liste-stil-resmi: Bu özellikleri liste için tanımlar.

Varsayılan değer yoktur

Liste Resim (list-image) Özelliği

Madde işaretini bir resimle değiştirmeye yarar.

Not: Resmin olmaması olasılığına karşın list-style-type özelliğini mutlaka tanımlayın.
 
ol
{
list-style-image: url(blueball.gif);
list-style-type: circle
}
Olası değerler:
url-adresi: Resmin Internet adresi
none: Varsayılan değer. Resim görüntülenmez

Liste Stil Yeri (list-style-position) Özelliği

Liste stil yeri madde işaretini listedeki yerini gösterir. 
ol
{
list-style-position: inside
}
Olası değerler:
inside: İşareti ve yazıyı içeri alır.
outside: Varsayılan değer. İşareti yazının soluna alır.

Liste Stil Tipi (list-style-type) Özelliği

ul
{
list-style-type: disc
}
Olası değerler:
none: İşaret yok
disc: Varsayılan değer. İşaret içi dolu çemberdir.
circle: İşaret çemberdir
square: İşaret karedir decimal: İşaret sayıdır
decimal-leading-zero: İşaret soldaki sıfırlarıyla sayıdır (01, 02, 03, vb.)
lower-roman: İşaret küçük harf roman rakamlarıdır (i, ii, iii, iv, v, vb.)
upper-roman: İşaret büyük harf Roman rakamlarıdır (I, II, III, IV, V, vb.)
lower-alpha: İşaret küçük harflerdir (a, b, c, d, e, vb.)
upper-alpha: İşaret büyük harflerdir (A, B, C, D, E, vb.)
lower-greek: İşaret küçük yunan harfleridir (alpha, beta, gamma, vb.)
lower-latin: İşaret küçük latin harflerdir (a, b, c, d, e, vb.)
upper-latin: İşaret büyük latin harflerdir (A, B, C, D, E, vb.)
hebrew: İşaret geleneksel Hebrew sayılarıdır
armenian: İşaret geleneksel ermeni sayılarıdır
georgian: İşaret geleneksel gürcü sayılarıdır
cjk-ideographic: İşaret düz ideografik sayılardır
hiragana: İşaret a, i, u, e, o, ka, ki, vb.
katakana: İşaret: A, I, U, E, O, KA, KI, vb.
hiragana-iroha: İşaret: i, ro, ha, ni, ho, he, to, vb.
katakana-iroha: İşaret: I, RO, HA, NI, HO, HE, TO, vb.
