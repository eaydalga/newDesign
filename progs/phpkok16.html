Değişmezler

Veritabanı ve Deyim nitelikleri tablosu

Nitelikler Geçerli değer(ler)
PDO_ATTR_AUTOCOMMIT BOOL
Eğer autocommit değeri varsa TRUE, aksi halde FALSE.
dbh->auto_commit değeri içerir. PDO'dan doğrudan gönderilir.
PDO_ATTR_PREFETCH LONG
Sürücüde onu destekleyen önalma (prefetch) değeri.
PDO_ATTR_TIMEOUT LONG
Zaman aşımından önce db işlemi için ne kadar bekleneceğini belirtir.
PDO_ATTR_ERRMODE LONG
PDO tarafından ele alınır ve işlenir
PDO_ATTR_SERVER_VERSION STRING
Bu sürücünün bağlı olduğu Sunucu/Sürüm bilgisini gösteren "insanın okuyabileceği" yazı dizisidir.
PDO_ATTR_CLIENT_VERSION STRING
Bu sürücünün desteklediği İstemci/Sürüm bilgisini içeren "insanın okuyabileceği" yazı dizisidir.
PDO_ATTR_SERVER_INFO STRING
Sunucunun "insanın okuyabileceği" açıklaması
PDO_ATTR_CONNECTION_STATUS LONG
Henüz tanımlanmamış değerler
PDO_ATTR_CASE LONG
PDO tarafından ele alma ve işleme.
PDO_ATTR_CURSOR_NAME STRING
Veritabanı imgeci için yazı dizisi adını gösterir. Bu ad "where current in <yazi_dizisi_adi>" SQL deyiminde kullanılır.
PDO_ATTR_CURSOR LONG
PDO_CURSOR_FWDONLY imleci ileri (Forward) yalnız ilerlet
PDO_CURSOR_SCROLL Kaydırılabilen imleç (cursor)
Yukarıdaki niteliklerin değerleri Zend API kurallarına göre tanımlanmıştır. *zval değerini çevirmede kullanılan makrolar Zend API içindedir. Bu makrolar Zend başlık dosyasında bulunur. PHP hazırlama yolundaki Zend yolunda zend_API.h dosyasıdır. Bu niteliklerin bazıları deyim niteliklerini ele almada kullanılabilir (PDO_ATTR_CURSOR ve PDO_ATTR_CURSOR_NAME gibi). Ayrıntılar için deyim niteliklerini ele alma işlevlerine bakın.
