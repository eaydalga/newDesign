Denemeler

PDO'nun çekirdek denemeleri vardır. Tüm sürücüler bunlardan geçebilmelidir. Yoksa sürüm olamazlar. PHP kaynak dağıtımından çalışırlar, o halde sizin sürücünüz için denemeleri çalıştırmak bazı şeyleri biraz çevirmeyi gerektirir. Önerilen işlem en son PHP 5.1 resim çekerini (snapshot) indirmek ve aşağıdaki işlemleri yapmaktır: 
$ cp -r pdo_SKEL /path/to/php-5.1/ext
Bu denemelerinize başlamanız için koşumları tutmanızı sağlar. Sonraki işlem PDO sıradan çekirdek denemeleri yaratmaktır. Genelde bu dosyaya common.phpt adı verilir. Bu dosya ext_skel tarafından yaratılan "tests" adlı dosya yoluna yerleştirilmelidir. Bu dosyanın içeriği aşağıdaki gibi olur: 
--TEST--
SKEL
--SKIPIF--
<?php # vim:ft=php
if (!extension_loaded('pdo_SKEL')) print 'skip'; ?>
--REDIRECTTEST--
if (false !== getenv('PDO_SKEL_TEST_DSN')) {
# user set them from their shell
   $config['ENV']['PDOTEST_DSN'] = getenv('PDO_SKEL_TEST_DSN');
   $config['ENV']['PDOTEST_USER'] = getenv('PDO_SKEL_TEST_USER');
   $config['ENV']['PDOTEST_PASS'] = getenv('PDO_SKEL_TEST_PASS');
   if (false !== getenv('PDO_SKEL_TEST_ATTR')) {
      $config['ENV']['PDOTEST_ATTR'] = getenv('PDO_SKEL_TEST_ATTR');
   }
   return $config;
}
return array(
   'ENV' => array(
           'PDOTEST_DSN' => 'SKEL:dsn',
           'PDOTEST_USER' => 'username',
           'PDOTEST_PASS' => 'password'
       ),
   'TESTS' => 'ext/pdo/tests'
   );
Bu sıradan çekirdek denemeleri çalıştırmanızı sağlar. PDOTEST_DSN, PDOTEST_USER ve PDOTEST_PASS değerlerini PDO yapıcısına dsn olarak geçirilir. Kullanıcı kodu ve şifresini girilir. Önce çevre incelenir, böylece gerekli değerler deneme ortamına taşınabilir.

Test koşumları aşağıdaki gibi çalıştırılabilir: 
$ cd /path/to/php-5.1
$ make TESTS=ext/pdo_SKEL/tests PDO_SKEL_TEST_DSN="skel:dsn" \
 PDO_SKEL_TEST_USER=user PDO_SKEL_TEST_PASS=pass test
