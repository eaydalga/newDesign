DEA (DATA ENCRYPTION ALGORITHM)

Bu program mantığı şifreleme işlemlerinde kullanılabilen DES algoritmasını açıklar.

DEAi, aşağıdaki çizimde gösterilen akışı onaltı kez yapar.

genel dea program mantığı : 

işlev girdileri : 
    encrypt/decrypt, input data, key, input data length.
işlev bilgi alanları :
    k0 k1 each 64 bits in length
    lr0 64 bits in length
    lr1 96 bits in length
    l0 r0 each 32 bits in length
    l1 r1 each 48 bits in length
calculate key from K0 -> C1,D1
calculate L0,R0 from input
counter = 0;
do {
   calculate K1
   calculate R1 as L0 + f(R0,K1)
   counter++;
   } while(counter < 16);
apply inverse permutation to L1 and R1.
