Paketleme ve dağıtım

Paket Yaratma

PDO sürücüleri PECL ile dağıtılır. PECL uzantılarının tüm genel kuralları uygulanır. Paket geçerli package.xml dosyası yaratılarak hazırlanır ve çalıştırılır. 
$ pecl package
Bu PDO_SKEL-X.Y.Z.tgz adında bir tar-topu yaratır.

Before releasing the package, you should test that it builds correctly; if you've made a mistake in your config.m4 or package.xml files, the package may not function correctly. You can test the build, without installing anything, using the following invocation: Paketin sürümünü yayımlamadan doğru kurulduğunu denemelisiniz. Eğer config.m4 dosyanızda bir hata yapmışsanız paketiniz doğru işev yapmaz. Paketi hazırlamayı kurmadan deneyebilirsiniz. Aşağıdaki çağırma işlemi bunu yapar: 
$ pecl build package.xml
Bir kez bunun çalışacağı ispatlanınca, kurulumu deneyebilirsiniz. 
$ pecl package
$ sudo pecl install PDO_SKEL-X.Y.X.tgz
package.xml için tüm ayrıntılar PEAR programcısının belgelerinde bulabilirsiniz (http://pear.php.net/manual).

Paketi Yayımlama

PDO sürücüsü PECL ile yayımlanır. iPECL hakkında bilgi http://pecl.php.net/index.php adresinden sağlanabilir.
