HTML 4.0 ve üzeri

HTML dosyalarını HTML 4.01 için denetletme

HTML sayfaları Döküman Tipi Tanımıyla (Document Type Definition, DTD) ile denetlenir. Denetlemenin doğru yapılabilmesi için doğru DTD tanımı dosyaya ilk satır olarak eklenmelidir.

"HTML 4.01 Strict DTD" denetimi eskimemiş ve "frameset" içinde yer almamış öğe ve nitelikleri içerir.

<!DOCTYPE HTML PUBLIC
"-//W3C//DTD HTML 4.01//EN"
"http://www.w3.org/TR/html4/strict.dtd">

"HTML 4.01 Transitional DTD" strict DTD'nin içerdiği herşeyi içerir, ek olarak eskimiş öğe ve nitelikler de cenetlenir.

<!DOCTYPE HTML PUBLIC
"-//W3C//DTD HTML 4.01 Transitional//EN"
"http://www.w3.org/TR/html4/loose.dtd">

"HTML 4.01 Frameset DTD" Transitional DTD'nin içerdiği herşeyi içerir ek olarak "çerçeve" denetimi de yapar.

<!DOCTYPE HTML PUBLIC
"-//W3C//DTD HTML 4.01 Frameset//EN"
"http://www.w3.org/TR/html4/frameset.dtd">
