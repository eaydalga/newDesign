MySQL İşlevleri (PDO_MYSQL)

Giriş

PDO_MYSQL, PHP Veri nesneleri (PDO) arayüzünün MySQl 3.x, 4.x ve 5.x veritabanına erişimini sağlayan bir sürücüdür.

PDO_MYSQL, MySQL 4.1 ve daha üstünde desteklenen yerel deyim hazırlığının üstünlüğünü kullanır. Eğer daha eski sürümler kullanıyorsanız mysql istemci kitaplığındaki eksikliği PDO benzetimle giderir.

Uyarı

Dikkat: Bazı MySQL tablo tipleri (sakalam makineleri) işlemleri desteklemez. İşlemsel veritabanı kodu yazarken kullanılan tablo türünde işlem desteklenmezse MySQL bir işlemin başarıyla başlatıldığını varsayar. Ek olarak başlatılan herhangi bir DDL sorgusunun beklemesi durumunda içsel onayı verilir.

Önceden Tanımlı Değişkenler

Aşağıdaki değişmezler bu sürücü için tanımlanmıştır. Uzantı kullanıldığında anlamı vardır. Eğer mysqle özel nitelikleri postgres sürücüyle kullanırsanız beklenmedik sonuçlar elde edebilirsiniz. PDO::getAttribute() PDO_ATTR_DRIVER_NAME niteliğini elde etmek için kullanılabilir. Böylece sürücü denetlenmiş olur (Eğer çoklu sürücüyle çalışan kodlamanız varsa önemlidir).
PDO::MYSQL_ATTR_USE_BUFFERED_QUERY (tamsayı)
Eğer bu nitelik PDOStatement üzerinde TRUE olursa, o zaman iMySQL sürücüyle MySQL API tamponlu sürümü kullanılmış olur. Eğer taşınabilir kod yazıyorsanız PDOStatement::fetchAll() kullanmalısınız.

Örnek-1 mysql içinde sorguları tamponlamaya zorlama 
    <?php
    if ($db->getAttribute(PDO::ATTR_DRIVER_NAME) == 'mysql') {
        $stmt = $db->prepare('select * from foo',
            array(PDO::MYSQL_ATTR_USE_BUFFERED_QUERY => true));
    } else {
        die("my application only works with mysql; I should use \$stmt->fetchAll() instead");
    }
    ?>
PDO::MYSQL_ATTR_LOCAL_INFILE (tamsayı)
LOAD LOCAL INFILE açma.
PDO::MYSQL_ATTR_INIT_COMMAND (tamsayı)
MySQL sunucuya bağlanırken kullanılacak komut. Yeniden bağlanırken otomatik olarak yeniden çalıştırılır.
PDO::MYSQL_ATTR_READ_DEFAULT_FILE (tamsayı)
my.cnf yerine seçenekleri adlandırılan dosyadan oku.
PDO::MYSQL_ATTR_READ_DEFAULT_GROUP (tamsayı)
Seçenekleri my.cnf ya da MYSQL_READ_DEFAULT_FILE ile belirtilen dosyadan adlandırılan grup için oku
PDO::MYSQL_ATTR_MAX_BUFFER_SIZE (tamsayı)
EN büyük tampon alan boyu. Varsayılan 1 MiB.
PDO::MYSQL_ATTR_DIRECT_QUERY (tamsayı)
Doğrudan sorguları yap. Hazırlanan deyimleri kullanma.
