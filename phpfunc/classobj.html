Sınıf/Nesne (Class/Object) İşlevleri

Giriş

Bu işlev sınıflar ve nesne kopyaları hakkında bilgi toplamanızı sağlar. Bir nesnenin ait olduğu sınıfın adı öğrenebilirsiniz. Bu işlevleri kullanarak, yalnız nesnenin sınıf üyeliğini bulmaz, babalık bilgilerine de erişebilirsiniz.

Örnekler

Bu örneklerde önce temel sınıf tanımlanmış daha sonra uzantı eklenmiştir. Temel sınıf genel olarak vegetable (sebze) tanımını içerir (değiştirilebilir mi ve rengi nedir gibi). Alt sınıf Spinach (Ispanak) ona cook (pişirme) adlı yöntem ekler ve pişip pişmediğini belirtir.

Örnek: classes.inc

<?php

// member özellikleri ve yöntemleriyle temel sınıf
class Vegetable {

    var $edible;
    var $color;

    function Vegetable($edible, $color="green") 
    {
        $this->edible = $edible;
        $this->color = $color;
    }

    function is_edible() 
    {
        return $this->edible;
    }

    function what_color() 
    {
        return $this->color;
    }
    
} // Vegetable sınıfı sonu

// temel sınıfın uzantısı
class Spinach extends Vegetable {

    var $cooked = false;

    function Spinach() 
    {
        $this->Vegetable(true, "green");
    }

    function cook_it() 
    {
        $this->cooked = true;
    }

    function is_cooked() 
    {
        return $this->cooked;
    }
    
} // Spinach sınıf sonu

?>
We then instantiate 2 objects from these classes and print out information about them, including their class parentage. We also define some utility functions, mainly to have a nice printout of the variables.

Örnek: test_script.php

<?php

include "classes.inc";

// yardımcı işlevler

function print_vars($obj) 
{
    foreach (get_object_vars($obj) as $prop => $val) {
        echo "\t$prop = $val\n";
    }
}

function print_methods($obj) 
{
    $arr = get_class_methods(get_class($obj));
    foreach ($arr as $method) {
        echo "\tfunction $method()\n";
    }
}

function class_parentage($obj, $class) 
{
    if (is_subclass_of($GLOBALS[$obj], $class)) {
        echo "$obj nesnesi " . get_class($$obj). " sınıfınındır";
        echo " $class alt sınıfıdır\n";
    } else {
        echo "$obj nesnesi $class alt sınıfına ait değildir\n";
    }
}

// iki nesnenin kopyaları

$veggie = new Vegetable(true, "blue");
$leafy = new Spinach();

// nesneler hakkında bilgi yazdır
echo "veggie: CLASS " . get_class($veggie) . "\n";
echo "leafy: CLASS " . get_class($leafy);
echo ", PARENT " . get_parent_class($leafy) . "\n";

// veggie özelliklerini göster
echo "\nveggie: Özellikleri\n";
print_vars($veggie);

// ve leafy yöntemlerini
echo "\nleafy: Yöntemler\n";
print_methods($leafy);

echo "\nBabalık:\n";
class_parentage("leafy", "Spinach");
class_parentage("leafy", "Vegetable");
?>
One important thing to note in the example above is that the object $leafy is an instance of the class Spinach which is a subclass of Vegetable, therefore the last part of the script above will output: 
       [...]
Bablık:
leafy nesnesi Spinach alt sınıfına ait değildir
leafy nesnesi spinach sınıfınındır Vegetable alt sınıfıdır

İçindekiler

  • call_user_method_array — artık kullanılmıyor
  • call_user_method — artık kullanılmıyor
  • class_exists — Sınıf tanımlandı mı diye bakar.
  • get_class_methods — Sınıfın yöntemlerinin adlarını alır
  • get_class_vars — Sınıfın varsayılan özelliklerini alır
  • get_class — nesne sınıfının adını döndürür.
  • get_declared_classes — Tanımlanan sınıfların adlarından oluşan diziyi döndürür.
  • get_declared_interfaces — Tüm tanımlanan arayüzlerin dizisini döndürür
  • get_object_vars — Verilen nesnenin özelliklerini alır
  • get_parent_class — Nesne y ada sınıfın üst düzey (babası) sınıf adını
  • döndürür
  • interface_exists — Arayüz tanımlı mı ona bakar.
  • is_a — Nesnenin bu sınıftan olup olmadığına bakar ya da bu sınıf üst düzeyden (babasından) biri olup olmadığına bakar.
  • is_subclass_of — Nesnenin bu sınıfı ailesinden (üst düzeyinden) biri
  • olup olmadığına bakar.
  • method_exists — "class" yönteminin olup olmadığına bakar.
  • property_exists —Nesne y ada sınıfın özelliği olup olmadığına bakar.
