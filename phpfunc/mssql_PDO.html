Microsoft SQL Sunucu ve Sybase İşlevleri(PDO_DBLIB)

Giriş

Uyarı

Bu uzantı DENEYSELDİR. Bu uzantının davranışları, işlevlerin değişgeçleri ve hatta adları bile değişebilir. PDO_DBLIB sürücüsü PHP Veri Nesnelerini (PDO) arayüzünü uygular. Böylece PHP içinden Microsoft SQL sunucu veritabanına FreeTDS kitaplığıyla erişilebilir.

Windows ortamında, PDO_ODBC sürücüsü Microsoft SQL Sunucuya ve Sybase veritabanına bağlantı için kullanılır.
