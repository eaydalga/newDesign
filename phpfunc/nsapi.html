PHP Hyperlink Preprocessor

15 Nisan 2008

NSAPI-özel işlevleri

Giriş

Bu işlevler PHP NSAPI modülü olarak Netscape/iPlanet/Sun web sunucularında çalıştığında geçerlidir.

Kurulum

PHP kurulumunun Netscape/iPlanet/Sun web sunucularındaki işlemleri için NSAPI bölümüne bakın (UNIX, Windows).

Çalışma Zamanı Ayarları

Bu PHP modulünün davranışı php.ini ayaralrından etkilenir. php.ini içindeki Seçenek ayarları obj.conf içindeki php4_execute çalıştırılarak da değiştirilebilir.

Ayrıca şunlara bakın

NSAPI Apache modülünden işlevlerin alt kümenin uygulanması ve en yüksek uyumun sağlanmasına çalışılır.

Apache işlevlerinin NSAPI tarafından uygulanması

Apache işlevi (yalnız eşdeğer olarak)NSAPI İşleviAçıklama
apache_request_headers()nsapi_request_headers()HTTP istek başlıklarının hepsini bulur
apache_response_headers()nsapi_response_headers()HTTP yanıt başlıklarının tümünü bulur
getallheaders()nsapi_request_headers()HTTP istek başlıklarının hepsini bulur
virtual()nsapi_virtual()NSAPI alt-isteği oluşturma

İçindekiler

  • nsapi_request_headers — HTTP istek başlıklarının hepsini bulur
  • nsapi_response_headers — HTTP yanıt başlıklarının hepsini bulur
  • nsapi_virtual — bir NSAPI alt-isteği yapar
