Tidy İşlevi

Giriş

Tidy HTML temizleme ve tamir hazır programına bir bağlantıdı. Yalnız HTML belgelerini temizlemek için değil, aynı zamanda belge ağacını dolaşmak için de kullanılır.

Gereksinim

Tidy kullanmak için libtidy kitaplığının kurulmuş olması gerekir. Ayrıntılı bilgi için http://tidy.sourceforge.net/ adresine bakın.

Kurulum

Tidy şu anda PHP 4.3.x ile kullanılabilir ve PHP 5'de PECL uzantısı olarak kullanıma sunulmuştur. Bakınız: http://pecl.php.net/package/tidy.

Not: Tidy 1.0 Yalnız PHP 4.3.x içindir. Ama Tidy 2.0 yalnız PHP 5 için hazırlanmıştır.

Eğer PEAR *nix-benzeri sisteminizde varsa pear kurucusunu kullanabilirsiniz. Kullanılacak komut: 
pecl install tidy
Her zaman tar..gz paketini indirip elle de kurulum yapabilirsiniz:

Örnek-1 PHP 4.3.x sürümünde elle tidy kurulumu

gunzip tidy-xxx.tgz
tar -xvf tidy-xxx.tar
cd tidy-xxx
phpize
./configure && make && make install
Wİndows kullanıcıları http://pecl.php.net/package/php_tidy.dll kitaplığını indirip kullanabilirler.

PHP 5'de "--with-tidy" seçeneğiyle derleme yapabilirsiniz.

Önceden Tanımlı Sınıflar

tidyNode
Yöntemler (Methods)
  • tidyNode::getParent - O anki düğümün babasını döndürür
  • tidyNode->hasChildren - o anki düğümün çocukları varsa TRUE döndürür
  • tidyNode->hasSiblings - o anki düğümün kardeşleri varsa TRUE döndürür
  • tidyNode->isAsp - o anki düğüm ASP düğümüyse TRUE döndürür
  • tidyNode->isComment - o anki düğüm açıklamaysa TRUE döndürür
  • tidyNode->isHtml - O anki düğüm HTML ise TRUE döndürür
  • tidyNode->isJste - o anki düğüm JSTE ise TRUE döndürür
  • tidyNode->isPhp - o anki düğüm PHP ise TRUE döndürür
  • tidyNode->isText - o anki düğüm yazı ise TRUE döndürür
Özellikler (Properties)
  • value - düğümün değeri (örneğin html yazı)
  • name - biçimin adı (örneğin html, a, vb..)
  • type - düğümün tipi (yukarıdaki değişmezlerden biri, örneğin TIDY_NODETYPE_PHP)
  • line* - düğümün başladığı yerdeki satır
  • column* - düğümün başladığı yerdeki kolon
  • proprietary* - TRUE eğer düğüm özel biçimi gösteriyorsa
  • id - biçimin tanıtım kodu (yukarıdaki değişmezlerden biri, örneğin TIDY_TAG_FRAME)
  • attribute - o anki düğümün niteliklerinden oluşan bir dizi ya da hiç nitelik yoksa NULL
  • child - çocuk tidyNode'ların dizisi, ya da NULL eğer hiç bir yoksa
Not: * ile işaretlenen özellikler PHP 5.1.0 ile vardı.

Örnekler

Bu basit örnek tidy kullanımını gösterir.

Örnek-2 Temel Tidy kullanımı

<?php
ob_start();
?>
<html>a html document</html>
<?php
$html = ob_get_clean();

// Belirtilen ayar
$config = array(
           'indent'         => true,
           'output-xhtml'   => true,
           'wrap'           => 200);

// Tidy
$tidy = new tidy;
$tidy->parseString($html, $config, 'utf8');
$tidy->cleanRepair();

// Çıktı
echo $tidy;
?>

İçindekiler

  • ob_tidyhandler — tamponu tamir etmek için ob_start geri alma işlevi
  • tidy_access_count — Belirtilen belgede Tidy erişilebilir uyarıların sayısını döndürür
  • tidy_clean_repair — Execute configured cleanup and repair operations on parsed markup
  • tidy_config_count — Returns the Number of Tidy configuration errors encountered for specified document
  • tidy::__construct — Constructs a new tidy object
  • tidy_diagnose — Run configured diagnostics on parsed and repaired markup
  • tidy_error_count — Returns the Number of Tidy errors encountered for specified document
  • tidy_get_body — tidy tarama ağacının ilk biçiminden başlayarak tidyNode nesnesini döndürür
  • tidy_get_config — O anki tidy ayarlarını al
  • tidy_get_error_buffer — belirtilen belgeyi tararken oluşan hata ve uyarıları döndürür
  • tidy_get_head — tidy tarama ağacının ilk biçiminden başlayarak tidyNode nesnesini döndürür
  • tidy_get_html_ver — Belirtilen belge için bulunan HTML sürümünü al
  • tidy_get_html — tidy tarama ağacının ilk biçiminden başlayarak tidyNode nesnesi döndürür.
  • tidy_get_opt_doc — verilen seçenek adı için belgeyi döndürür
  • tidy_get_output — taranan tidy belgesini (markup) gösteren yazı dizisini döndürür
  • tidy_get_release — Tidy kitaplığının sürüm tarihini al
  • tidy_get_root — tidy tarama ağacının kökünü gösteren tidyNode nesnesini döndürür
  • tidy_get_status — Belirtilen belgenin durumunu al
  • tidy_getopt — tidy belgesi için belirtilen ayar değerini döndürür
  • tidy_is_xhtml — Belgenin XHTML belgesi olup olmadığını belirtir
  • tidy_is_xml — belgenin soysal (HTML/XHTML değil) olup olmadığını belirtir
  • tidy_load_config — Belirtilen kodlamayla ASCII Tidy ayar dosyasını yükler
  • tidy_node->get_attr — Sağlanan nitelik tanıtımıyla niteliği döndürür
  • tidy_node->get_nodes — Belirtilen tanıtımla bu düğüm altındaki düğümlerin dizisini döndürür
  • tidy_node->next — Bu düğümün sonraki kardeşini döndürür
  • tidy_node->prev — Bu düğümün önceki kardeşini döndürür
  • tidy_parse_file — dosyadaki ya da URI'deki biçimleri (markup) tara
  • tidy_parse_string — Bir yazı dizisindeki belgeyi tara
  • tidy_repair_file — Bir dosyayı tamir et ve yazı dizisi olarak döndür
  • tidy_repair_string — Seçenek olarak sağlanan ayar dosyasını kullanarak yazı dizisini tarama
  • tidy_reset_config — Tidy ayarlarını varsayılanlara döndür.
  • tidy_save_config — O anki ayarları adlandırılan dosyaya yaz
  • tidy_set_encoding — Giriş/Çıkış karakter kodlamasını biçim (markup) taraması için ayarla
  • tidy_setopt — belirtilen tidy belgesinin ayarlarını günceller
  • tidy_warning_count — Belirtilen belgenin Tidy uyarı sayısını döndürür
  • tidyNode->hasChildren — Bu düğümün çocukları varsa true döndürür
  • tidyNode->hasSiblings — düğümün kardeşleri varsa true döndürür
  • tidyNode->isAsp — Bu düğüm ASP ise true döndürür
  • tidyNode->isComment — Düğüm bir açıklamayı belirtiyorsa true döndür
  • tidyNode->isHtml — Bu düğüm HTML belgesinin bir parçasıysa true döndürür
  • tidyNode->isJste — Eğer düğüm JSTE ise true döndürür
  • tidyNode->isPhp — Eğer düğüm PHP ise true döndürür
  • tidyNode->isText — Eğer bu düğüm yazıyı (biçim değil) gösterirse true döndürür
  • tidyNode::getParent — o anki düğümün babasını döndürür
