MaxDB PHP Uzantısı

Giriş

MaxDB PHP uzantısı 7.5.0 ve üzeri sürümlerde sağlanan işlevselliğe erişmenizi sağlar. Daha çok bilgi için http://www.mysql.com/products/maxdb/ adresine bakın.

MaxDB PHP uzantısı MySQL mysqli uzantısıyla uyumludur. MaxDb ve MySQL verştabanı sunucularının farklarından ikisi arasında çok küçük ayrıcalıklar bulunur.

Ana değişiklikler şunlardır:

  • maxdb_character_set_name() - Yalnız ascii ya da unicode döndürür
  • maxdb_get_client_info() - Değişik sürüm yazı dizisi döndürür
  • maxdb_get_client_version() - Değişik sürüm yazı dizisi döndürür
  • maxdb_get_host_info() - Yerel bilgisayar ya da bilgisayar adı döndürür
  • maxdb_get_server_info() - Değişik sürüm yazı dizisi döndürür
  • maxdb_get_server_version() - Değişik sürüm yazı dizisi döndürür
  • maxdb_kill() - Yalnız oturumu kapatır
  • maxdb_multi_query() - Birden çok SQL deyimini ele alamaz
  • maxdb_next_result() - İşlev her zaman False döndürür
  • maxdb_options() - Başka tür seçenek kümesini destekler
  • maxdb_report() - Farklı bir rapor konumunu destekler
  • maxdb_stat() - Farklı bir sistem durum kodu yazı dizisini döndürür
  • maxdb_stmt_store_result() - MaxDB için gerekli değildir
  • maxdb_store_result() - MaxDb için gerekli değildir.
MaxDB için gereken belgeler http://dev.mysql.com/doc/maxdb/ adresinde bulunabilir.

Gereksinim

Bu işlevlerin kullanılabilir olması için PHP MaxDB desteğiyle derlenmelidir. Ek olarak MaxDB SQLDBC çalışma zamanı kitaplığı olmalı ki MaxDB sunucusunu kullanabilesiniz.

MaxDB SQLDBC için belgeler http://dev.mysql.com/doc/maxdb/ adresinde bulunabilir.

MaxDB SQLDBC paketini http://dev.mysql.com/downloads/maxdb/clients.html adresinden indirebilirsiniz.

Kurulum

"--with-maxdb[=DIR]" ayar seçeneğini kullanarak PHP'nin MaxDB erişimini açmış olursunuz. [DIR] MaxDB SQLDBC paketinin kurulu olduğu yolu verir.

Windows users will need to enable php_maxdb.dll inside of php.ini.

Örnekler

Tüm MaxDB PHP belgelerindeki örnekler MaxDB'deki HOTELDB demo veritabanını kullanır. http://dev.mysql.com/doc/maxdb/en/98/11b83fa6b33c17e10000000a114084/frameset.htm adresinde bu bilgiyi bulabilirsiniz.

MaxDB PHP belgelerindeki örnekleri kullanabilmek için veritabanına eğitim bilgilerini yüklemeniz gerekir. Ayrıca php.ini içindeki maxdb.default_db değerini ayarlamış olmalısınız.

Aşağıdaki basit örnek nasıl bağlantı kurulacağını, nasıl sorgulama çalıştırılacağını sonuçların nasıl yazdırılacağını ve nasıl MaxDB veritabanı bağlantısının kesileceğini gösterir.

Örnek-1 MaxDB uzantısı özet işlem örneği

<?php
$link = maxdb_connect("localhost", "MONA", "RED", "DEMODB");
   
/* bağlantıya bak */
if (maxdb_connect_errno()) {
   printf("Bağlantı kurulamadı: %s\n", maxdb_connect_error());
   exit();
}

/* SQL sorgu hazırlama */
$query = "SELECT * FROM hotel.city";
$result = maxdb_query($link, $query)
          or die("Sorgu olmadı : " . maxdb_error());

/* Sonuçları HTML olarak yazdırmak */
echo "\n";
while ($line = maxdb_fetch_array($result, MAXDB_ASSOC)) {
    echo "  \n";
    foreach ($line as $col_value) {
        echo "    \n";
    }
    echo "  \n";
}
echo "
$col_value
\n";

/* sonuçları boşaltmak */
maxdb_free_result($result);

/* Bağlantıyı kapamak */
maxdb_close($link);
?>
Aşağıdaki örnek değişkenlerin nasıl SELECT INTO deyimine bağlanacağını gösterir.

Örnek-2 SELECT INTO deyiminin kullanımına ilişkin örnek

<?php
$link = maxdb_connect("localhost", "MONA", "RED", "DEMODB");

/* bağlantıya bak */
if (!$link) {
   printf("Bağlantı kurulamadı: %s\n", maxdb_connect_error());
   exit();
}
   
/* SQL sorgusunu hazırlama */
$stmt = maxdb_prepare ($link,
    "SELECT percentage INTO ? FROM hotel.countrylanguage where language = ?");
if (!$stmt) {
  printf ("Hazırlama olmadı: %s\n", maxdb_error($link));
}

$name = "Mbundu";

maxdb_stmt_bind_param($stmt, 'ds', $percentage, $name);
maxdb_stmt_execute($stmt);

printf ("%f\n", $percentage);

maxdb_stmt_close ($stmt);
?>

Örnek-3 Veritabanı altprogramlarını kullanma örneği

<?php
$link = maxdb_connect("localhost", "MONA", "RED", "DEMODB");

/* bağlantıya bak */
if (!$link) {
   printf("Bağlantı kurulamadı: %s\n", maxdb_connect_error());
   exit();
}

maxdb_report (MAXDB_REPORT_OFF);
maxdb_query($link,"DROP DBPROC test_proc");
maxdb_report (MAXDB_REPORT_ERROR);

$query = "create dbproc test_proc (INOUT e_text char(72))
                 AS select * from SYSDBA.DUAL; fetch into :e_text;";

maxdb_query($link, $query);

/* SQL sorgu hazırlama */
$stmt = maxdb_prepare ($link, "CALL test_proc (?)");
if (!$stmt) {
  printf ("Hazırlık olmadı: %s\n", maxdb_error($link));
}

maxdb_stmt_bind_param($stmt, 's', $result);
maxdb_stmt_execute($stmt);

printf ("%s\n", $result);

maxdb_stmt_close ($stmt);
?>
