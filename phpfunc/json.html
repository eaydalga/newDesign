JSON İşlevleri

Giriş

Bu uzantı JavaScript Nesne Gösterilişi (JSON) veri değiştirme kalıbını uygular. kod açma JSON_checker (Douglas Crockford) taramasıyla sağlanır.

Gereksinim

PHP 5.2.0 sürümünden beri JSON PHP ile paketleniyor.

PECL uzantılarında bu konudan kurulum bilgisine erişilebilir. Diğer bilgiler için http://pecl.php.net/package/json adresine bakın.

İçindekiler

  • json_decode — JSON yazı dizisinin kodunu aç
  • json_encode — Bir değerin JSON gösterilimini döndürür
