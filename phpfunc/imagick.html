Imagick Resim Kitaplığı

Giriş

Uyarı

Bu uzantı deneyseldir. Bu nedenle herşeyi değişebilir. Kullanacaksanız değişeceğini bilerek kullanmanız gerekir.
Imagick yerel php uzantısıdır. Resimleri ImageMagick API kullanarak yaratılır ve değiştirilebilir.

ImageMagick® bitmap resim yaratmak, düzeltmek ve oluşturmak için kullanılan yazılım paketidir. Resimleri okur, çevirir ve çeşitli kalıplarda yazar (100'den çok). DPX, EXR, GIF, JPEG, JPEG-2000, PDF, PhotoCD, PNG, Postscript, SVG, ve TIFF bunlardan bazılarıdır.

Copyright 1999-2007 ImageMagick Studio LLC, kazanç gütmeyen bir organizasyondur resim konusundanki yazılımı herkese açık yapmıştır.

Örnek

Imagick resim işlemlerini PHP'de OO arayüzüyle çok kolaylaştırmıştır. Aşağıda küçük resim yapma örneği vardır:

Örnek-1 Imagick içinde küçük resim yarat

<?php

header('Content-type: image/jpeg');

$image = new Imagick('image.jpg');

// Eğer 0 en ve boy değiştirgeç olarak sağlanıyorsa
// görünüş oranı korunur
$image->thumbnailImage(100, 0);

echo $image;

?>
Imagick içinde desteklenen SPL ve diğer OO özellikleri kullanılarak, bir yol altındaki tüm dosyaları yeniden boyutlandırmak çok basittir (Büyük kamera resimlerinin toplu işlemler web'de görünebilecek biçime getirilmesi açısından kullanışlıdır). Burada yeniden boyulandırma kullanılmıştır:

Örnek-2 Yoldaki JPG resimlerin küçüklerin yapma 
<?php

$images = new Imagick(glob('images/*.JPG'));

foreach($images as $image) {

    // 0 sunmak, görünüm oranının korunmasını zorlar
    $image->thumbnailImage(1024,0);

}

$images->writeImages();

?>
