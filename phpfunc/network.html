PHP_INI_* değişmezleri için php.ini yönelticilerine bakın.

Burada ayarlama yönelticilerinin kısa açıklaması vardır.
define_syslog_variables mantıksal
Whether or not to define the various syslog variables (e.g. $LOG_PID, $LOG_CRON, etc.). Turning it off is a good idea performance-wise. At runtime, you can define these variables by calling define_syslog_variables().

Kaynak türleri

Bu uzantı bir dosya göstergesi tanımlar. fsockopen() ve pfsockopen() işlevlerden üretilir.

İçindekiler

  • checkdnsrr — Verilen Internet host adı ya da IP adresine göre ilgili DNS kayıtlarını kontrol et
  • closelog — Sistem kaydının bağlantısını kapat
  • debugger_off — PHP hata yazma (debugger) olanağını kapat
  • debugger_on — PHP hata yazma (debugger) olanağını aç
  • define_syslog_variables — Syslog ile ilişkili değişmezlerin tümünü ilkler
  • dns_check_record — checkdnsrr eşdeğeri
  • dns_get_mx — getmxrr eşdeğeri
  • dns_get_record — Sunucu adıyla DNS kaynak kayıtlarını bulma
  • fsockopen — Internet ya da Unix alan socket bağlantısını aç
  • gethostbyaddr — Internet IP adresinden Internet sunucu adını alma
  • gethostbyname — Internet sunucu adından IP numarasını almak
  • gethostbynamel — Verilen sunucu adıyla ilişkili IP adreslerinin listesini alır
  • getmxrr — verilen Internet sunucu adının ilişik MX kaydını alma
  • getprotobyname — Protokol adıyla ilişkili protokol sayısı
  • getprotobynumber — Protokol sayısının ilişkili protokol adı
  • getservbyname — Internet servis ve protokolüyle ilişkili port numarasını al
  • getservbyport — Port ve protokolle ilişkili Internet servisini al
  • header — ham HTTP başlığını gönder
  • headers_list — gönderilen yanıt başlıklarının listesini döndürür (ya da göndermeye hazır)
  • headers_sent — başlıkların gönderilmesinin kontrol edilmesidir.
  • inet_ntop — Paketli Internet adresini okunabilir gösterime çevirir
  • inet_pton — Okunabilir IP adresini paketli adrese çevirir
  • ip2long — Yazı dizisi içerinde bulunan noktalı Internet Protokolu adresinin 8IPv4) düzgün adrese çevrilmesidir.
  • long2ip — IPv4 Internet adresinin noktalı kalıptaki yazı dizisi içeriğine çevrilmesi
  • openlog — Sistem kayıdına bağlantı aç
  • pfsockopen — Sürekli Internet ya da Unix socket bağlantısını aç
  • setcookie — Bir cookie gönder
  • setrawcookie — cookie değerinde url kodlaması yapmadan bir cookie gönder
  • socket_get_status — stream_get_meta_data eşdeğeri
  • socket_set_blocking — stream_set_blocking eşdeğeri
  • socket_set_timeout — stream_set_timeout eşdeğeri
  • syslog — Sistem kayıt mesajı üretir.
