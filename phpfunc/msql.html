mSQL İşlevleri

Giriş

Bu işlemler mSQL veritabanı sunucusuna erişime izin verir. Daha çok bilgi için http://www.hughes.com.au/ adresine bakın.

Kurulum

Bu işlevlerin çalışabilmesi için PHP "--with-msql[=DIR]" seçeneğiyle derlenmelidir. DIR mSQL yazılımının kurulu olduğu yoldur. Varsayılanı /usr/local/msql3 olarak bilinir.

Not: Win32 kullanıcıları için Not
Bu uzantının çalışması için Windows sistemi PATH çevresel değişkeninin erişebildiği yerde DLL dosyaları bulunmalıdır. DLL'leri PHP klasörüne taşımakla da çalışır.

Çalışma zamanı Ayarları

php.ini ayarları işlevlerin çalışmasını etkiler.

Kaynak Türleri

mSQL modülünde ki kaynak türü vardır. İlki veritabanı bağlantısı için bağlantı tanıtıcısı, ikincisi sorgulama sonuçlarını tutan kaynaktır.

Önceden Tanımlı Değişmezler

Aşağıdaki değişmezler bu uzantı için tanımlanmıştır. Bu uzantı yüklü olduğunda kullanılabilir. 
MSQL_ASSOC (tamsayı)
MSQL_NUM (tamsayı)
MSQL_BOTH (tamsayı)

Örnekler

Bu örnek nasıl bağlantı kurulacağını, nasıl sorgu çalıştırılacağını, sonuç satırlarının nasıl yazdırılacağını ve mSQL veri tabanından bağlantının nasıl kapatılacağını gösterir.

Örnek-1 mSQL kullanım örneği

<?php
/* Bağlanma, veri tabanını seçme */
$link = msql_connect('localhost', 'username', 'password')
    or die('Bağlanamadı : ' . msql_error($link));

msql_select_db('database', $link)
    or die('Veri tabanına bağlanamadı');

/* SQL sorgu gönder */
$query = 'SELECT * FROM my_table';
$result = msql_query($query, $link) or die('Query failed : ' . msql_error());

/* Sonuçları HTML olarak yaz */
echo "\n";
while ($row = msql_fetch_array($result, MSQL_ASSOC)) {
    echo "\t\n";
    foreach ($row as $col_value) {
        echo "\t\t\n";
    }
    echo "\t\n";
}
echo "
$col_value
\n";

/* Sonuç takımını boşalt */
msql_free_result($result);

/* Bağlantıyı kapat */
msql_close($link);
?>
