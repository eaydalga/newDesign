Informix İşlevleri (PDO_INFORMIX)

Giriş

PDO_INFORMIX PHP Veri Nesnelerini )PDO) uygulayan bir sürücüdür. Bu arayüz PHP içinden informix varitabanına erişimi sağlamak için kullanılır.

Kurulum

PDO_INFORMIX uzantısını hazırlamak için Informix Client SDK 2.81 UC1 ya da daha üst sürümü PHP ile aynı sistemde kurulmuş olmalıdır. Informix istemcisi IBM Informix destek sitesinden sağlanabilir.
PDO_INFORMIX bir PECL uzantısıdır. PECL uzantıları kurulumunda ayrıtılar bulunabilir. "configure" komutunun Informix Client SDK başlık dosyaları ve kitaplıkların yerini gösteren biçimi aşağıda gösterilmiştir: 
   bash$ ./configure --with-pdo-informix=/path/to/SDK[,shared]

The configure command defaults to the value of the INFORMIXDIR environment variable. Scrollable cursors PDO_INFORMIX supports scrollable cursors; however, they are not enabled by default. To enable scrollable cursor support, you must either set ENABLESCROLLABLECURSORS=1 in the corresponding ODBC connection settings in odbc.ini or pass the EnableScrollableCursors=1 clause in the DSN connection string.
