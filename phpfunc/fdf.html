Veri Bilgi Giriş Kalıbı İşlevleri
Forms Data Format Functions

Giriş

Veri Bilgi Giriş Kalıbı PDF belgeleri içindeki Bilgi Girişini ele alan kalıptır. http://partners.adobe.com/asn/acrobat/forms.jsp URL adresideki belgeyi okumalısınız. Burada FDF'nin ne olduğu ve genelde nasıl kullanıldığı anlatılır.

FDF genel anlamda HTML bilgi giriş formlarına benzer. Fark temelde verinin nasıl sunucuya gönderildiği konusundadır. Gönderme düğmesine basılır (Bu aslında Veri bilgi giriş kalıbı olur) ve bilgi giriş formunun kendisi (taşınabilir belge kalıbındadır) gönderilir. Bu veriyi işlemek fdf işlevlerinin işidir. Ama fazlası da vardır. Bririsi var olan bir PDF formunu alır ve formu değiştirmeden içini veriyle doldurabilir. Bu durumda FDF formu yaratılmalıdır (fdf_create()). Form içindeki her giri alanına değer yazılır (fdf_set_value()) ve bu bilgi PDF formuna ilişkilendirilir (fdf_set_file()). Son olarak bu bilgi tarayıcıya mime tipi application/vnd.fdf olan belge olarak gönderilir. Tarayıcınıza bağlı "acrobat reader" program eklentisi Mime tipini anlar ve ilgili PDF formunu okuyup içini FDF belgesinden doldurur.

FDF-belgesine bir yazı düzenleyicisiyle bakıldığında, adı FDF olan katalog nesnesi bulunur. Bu tür nesne Alanlar, F, Durum vb bir dizi öğeden oluşur. En sık kullanılan öğe giriş alanlarını gösteren alanlardır. F içindeyse PDF belgesinin dosya adı vardır. Bu öğeler FDF belgesinde /F-Key ya da /Status-Key olarak adlandırılır. Bu öğeleri değiştirmek fdf_set_file() ve fdf_set_status() gibi işlevlerle yapılır. Alanlarsa fdf_set_value(), fdf_set_opt() vb işlevlerle değiştirilir.

Gereksinim

http://partners.adobe.com/asn/acrobat/forms.jsp adresinden FDF araç-kiti SDK (yazılım geliştirme kiti) gerekecektir. PHP 4.3.0 sürümü için en az SDK sürüm 5.0 olmalıdır. FDF araç-kiti kitaplığı yalnız ikili biçimdedir. Adobe yalnız Win32, Linux, Solaris ve AIX ortamlarını destekler.

Kurulum

PHP "--with-fdftk[=DIR]" ile derlenir.

Not: Eğer PHP'de fdftk desteğini ayarlarken sorunlar yaşarsanız fdftk.h ve libfdftk.so doğru yerde olduğuna bakın. Ayar kodu FDF SDK dağılımı dosya yolu yapısını ve genelde kullanılan DIR/include ile DIR/lib kavramını destekler. O halde ister dağıtımı doğrudan açıp kullanın, isterseniz paketi açıp dosyaları doğru doya yollarına elinizle taşıyın (söz gelimi /usr/local/include ve /usr/local/lib gibi). Sonra PHP ayarlarını "--with-fdftk=/usr/local" biçimde yazın.

Not: Win32 Kullanıcıları içindir. Bu uzantının çalışabilmesi için, Windows sistem yolunda (PATH) bazı DLL dosyalarının bulunması gerekir. Bunun için ilgili başlığı incelemek gerekir (Windows PATH çevre değişkenine PHP dosya yolunu nasıl ekleyebilirim). Aslında PHP altından DLL dosyalarını Windows sistem klasörüne taşımakla sorun çözülür ama önerilmez. Bu uzantı şu dosyanın PATH değişkeninde tanımlı olmasını ister: fdftk.dll

Kaynak Türü

fdf
Pekçok fdf işlevi ilk parametre olarak fdf kaynaklarına gereksinim duyar. fdf kaynağı fdf dosyasını açmak için bir tutaçtır. fdf kaynakları fdf_create(), fdf_open() ve fdf_open_string() kullanılarak elde edilir.

Örnekler

Aşağıdaki örnek form verilerinin gelişimini anlatır

Örnek#1 FDF belgesi gelişimi 
<?php
// Giriş yazı dizisinden uzantıyla sağlanan fdf açın
// pdf bilgi giriş formu birçok giriş bilgisi içerir:
//The pdf form contained several input text fields with the names
// adlar (names), öbek (volume), tarih (date), açıklama (comment),
// yayıncı (publisher), hazırlayan (prepareer) ve iki kontrol kutusu
// yayımcı göster ve hazırlayıcı göster (show_publisher ve show_preparer).
$fdf = fdf_open_string($HTTP_FDF_DATA);
$volume = fdf_get_value($fdf, "volume");
echo "Öbek alanının değeri '<b>$volume</b>'<br />";

$date = fdf_get_value($fdf, "date");
echo "Tarih alanının değeri '<b>$date</b>'<br />";

$comment = fdf_get_value($fdf, "comment");
echo "Açıklama alanının değeri '<b>$comment</b>'<br />";

if (fdf_get_value($fdf, "show_publisher") == "On") {
  $publisher = fdf_get_value($fdf, "publisher");
  echo "Yayıncı alanının değeri '<b>$publisher</b>'<br />";
} else
  echo "Yayıncı gösterimeyecek.<br />";

if (fdf_get_value($fdf, "show_preparer") == "On") {
  $preparer = fdf_get_value($fdf, "preparer");
  echo "Hazırlayan alanının değeri '<b>$preparer</b>'<br />";
} else
  echo "Hazırlayan gösterilmeyecek.<br />";
fdf_close($fdf);
?>
