Haru PDF İşlevleri

Giriş

PECL/haru uzantısı Haru Ücretsiz PDF Kitaplığı için bağlantı sağlar - PDF yaratmak için ücretsiz, ortamdan ortama taşınan, açık kaynak yazılım kitaplığıdır.

Kitaplığı burada bulabilirsiniz: http://libharu.sourceforge.net.

UYARI

Bu uzantı deneyseldir. Bu uzantının davranışı -- işlevlerinin adları da dahil olmak üzere -- her an değişebilir.

Gereksinim

PECL/haru kullanmak için önce libharu kurulmalıdır. PECL/haru libharu 2.0.8 sürümüyle denenmiştir. Eski sürümler çalışmayabilir. PECL/haru PHP 5.1.3 sürümü ve sonrasını ister.

Kurulum

PECL uzantısının kurulumu için bilgi PECL uzantıları kurulumları adı belgelerde bulunur. Yeni sürünler gibi bilgiler için http://pecl.php.net/package/haru incelenmelidir. En son PECL/haru Win32 DLL kitaplığı: php_haru.dll.

Örnekler


Örnek1: Temel PECL/haru örneği
<?php

$doc = new HaruDoc;

$doc->setPageMode(HaruDoc::PAGE_MODE_USE_THUMBS); /* küçük resim göster*/

$page = $doc->addPage(); /* belgeye sayfa ekle */
$page->setSize(HaruPage::SIZE_A4, HaruPage::LANDSCAPE);
       /* sayfayı uzunlamasına A4 kalıbında ayarla */

$courier = $doc->getFont("Courier-Bold"); 
       /* paketlenmiş yazı tipi aşağıdaki satırlarda bulunur */

$page->setRGBStroke(0, 0, 0); /* renkleri ayarla */
$page->setRGBFill(0.7, 0.8, 0.9);
$page->rectangle(150, 150, 550, 250); /* dikdörtgen çizin */

$page->fillStroke(); /* doldur ve çiz */

$page->setDash(array(3, 3), 0);
       /* bu sayfadaki satırlar  çizgi stilini ayarlama */
$page->setFontAndSize($courier, 60); /* yazı tipi ve boyutunu belirleme */

$page->setRGBStroke(0.5, 0.5, 0.1); /* satır rengini belirleme */
$page->setRGBFill(1, 1, 1); /* doldurma renklerini belirle */

$page->setTextRenderingMode(HaruPage::FILL_THEN_STROKE);
       /* yazıyı doldur ve çiz */

/* yazıyı yazdır */
$page->beginText();
$page->textOut(210, 270, "Merhaba Dünya!");
$page->endText();

$doc->save("/tmp/test.pdf");
       /* belgeyi dosyaya yaz */

?>
Sonuç belgeyi sık kullandığınız PDF göstericisiyle açın. Açık mavi dikdörtgen ortasında beyazla yazılmış "Merhaba Dünya" bulursunuz.

Hazır yazı tipleri ve kodlaması

Hazır yazı tipleri (Fonts)

Base14 yazı tipleri PDF içine gömülmüştür ve tüm göstericiler onları görüntüleyebilir. Bu yazı tiplerini kullanarak sonuç dosyasının boyunu azaltabilir ve işlemeyi daha da hızlandırır. Dışsal yazı tiplerinin yüklenmesinden vaz geçilir. Yalnız, bu yazı tipleri "latin1" karakter takımını destekler, dışsal yazı tiplerini başka karakter takımları için yüklemeniz gerekecektir.

Base14 yazı tipleri:
  • Courier
  • Courier-Bold
  • Courier-Oblique
  • Courier-BoldOblique
  • Helvetica
  • Helvetica-Bold
  • Helvetica-Oblique
  • Helvetica-BoldOblique
  • Times-Roman
  • Times-Bold
  • Times-Italic
  • Times-BoldItalic
  • Symbol
  • ZapfDingbats
